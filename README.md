# 没猫病日志

[![standard-readme compliant](https://img.shields.io/badge/readme%20style-standard-brightgreen.svg?style=flat-square)](https://github.com/RichardLitt/standard-readme)

能实现日历式的治疗记录，关注每一次体重变化及治疗情况，方便根据数据统计来方便治疗。

## Background

方便猫传腹治疗记录

## Install

```sh
$ npm install --global
```

## Usage

```sh
$ php artisan serve
```
